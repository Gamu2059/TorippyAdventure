----とりっぴーアドベンチャー----

----起動方法----
grp05ディレクトリ直下にある[main.exe]を実行すること

----puttyの設定----

フォントサイズ4 (タブにはないためキーボードから直接入力すること)
フルスクリーンで実行すること。

----操作方法----

オートプレイ						:SHIFTキー + \キー(キーボードの右上の￥キ－)
スコアアタック&レコードプレイ		:SHIFTキー + @キー(要するに'`')
ノーマルプレイ						:上記以外のキー

カーソルキー						:移動(左右のみ)
Cキー								:攻撃(3段階まで連携可能)
Xキー								:ジャンプ(地上からなら2段可能)
Zキー								:回避
Zキー後に矢印キー					:敵の背後に回れる。
Ctrl+Cキー							:ゲームオーバー(強制終了ではない。1回やってみ)

壁に触れていて降下中にXキー			:壁キックジャンプ
CHARGEゲージMAX時 Shift+Cキー		:チャージアタック(衝撃波)
斜めジャンプ						:カーソルキーを押したままXキー

*他のPCでスコアアタックをプレイしている間はスコアアタックはプレイできません。
